# Components Monorepo

webman tech 组件集合仓库，包含多个可独立使用的 PHP 组件。

## 安装

```bash
# 安装所有组件
composer require webman-tech/components-monorepo

# 按需安装单个组件
composer require webman-tech/amis-admin
composer require webman-tech/auth
# ...
```
